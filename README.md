# 👋 Portofolio Rizki

Selamat datang di website portofolio profesional saya! Repository ini berisi kode sumber dari website portofolio personal yang menampilkan project, skill, pengalaman, dan pencapaian saya di bidang pengembangan web dan teknologi.

## 🌐 Website Live
Kunjungi portfolio saya di: [https://codrizsain.github.io/portofolio-rizki/](https://codrizsain.github.io/portofolio-rizki/)

## 📋 Tentang Portfolio Ini

Website portofolio ini dirancang untuk memberikan gambaran lengkap tentang:
- **Profil Profesional** - Informasi pribadi dan latar belakang karir
- **Project & Portfolio** - Showcase project yang telah saya kerjakan
- **Skill & Keahlian** - Teknologi dan tool yang saya kuasai
- **Pengalaman Kerja** - Riwayat pekerjaan dan pencapaian
- **Pendidikan** - Latar belakang pendidikan dan sertifikasi
- **Kontak** - Cara untuk menghubungi saya

## 🛠️ Teknologi yang Digunakan

- **HTML5** - Struktur dan markup website
- **CSS3** - Styling dan responsive design
- **JavaScript** - Interaktivitas dan fungsionalitas dinamis
- **GitHub Pages** - Hosting dan deployment

## 🚀 Cara Menjalankan Locally

1. Clone repository ini:
```bash
git clone https://github.com/codrizsain/portofolio-rizki.git
```

2. Masuk ke direktori project:
```bash
cd portofolio-rizki
```

3. Buka file `index.html` di browser Anda atau gunakan live server
